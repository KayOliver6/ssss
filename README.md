## ASDF Utility

A simple utility for keyboard layout testing and text pattern generation.

### Features

- Generate 'asdf' keyboard patterns
- Get the reverse pattern ('fdsa')
- Display the QWERTY keyboard home row

### Usage

```bash
python3 asdf.py
```

### Functions

- `asdf_pattern(length)`: Generate the 'asdf' pattern repeated `length` times
- `asdf_reverse()`: Return the reverse pattern 'fdsa'
- `asdf_keyboard_row()`: Return the home row of a QWERTY keyboard
